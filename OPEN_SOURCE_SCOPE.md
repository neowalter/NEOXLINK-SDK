# Open-Source Scope and Boundaries

## Included in this repository

- natural language intent parsing framework
- prompt templates and prompt evaluation tools
- data extraction and structuring pipeline
- taxonomy mapping system
- generic matching pipeline without proprietary weights
- data schema definitions
- connectors for public data sources
- data validation and cleaning tools
- contribution workflow and review tools

## Excluded from this repository

- production-grade provider database
- ranking weights derived from private real usage
- anti-abuse and spam detection logic
- submission view/satisfaction reward mechanisms and referral traffic incentives
- private or licensed data sources
- user-specific interaction history
- monetization logic

## Core principle

Open-source the mechanism, not the advantage.
